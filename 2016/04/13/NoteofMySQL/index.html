MySQL学习笔记

发表于 2016-04-13   |   分类于 MySQL   |  

简介

MySQL是一种DBMS，即数据库管理软件。MySQL在世界范围内已经得到了广泛的认可。MySQL作为数据库知识入门有诸多优点：MySQL是开源的，可以免费使用（甚至可以免费修改），搭建MySQL环境比较简单（可以在shell里做各种实验）。
关于数据库设计的知识，这里不做介绍，现在仅仅记录MySQL的基本操作。

MySQL的基本操作

连接数据库

在Shell里输入mysql -u root -p，输入密码后即可以root身份连接到MySQL，当然你也可以新建一个非root用户名来连接（实际应用中最好是这样，毕竟root权限比较大）。

操作数据库

输入USE databasename就可以选择使用名为databasename的数据库了。
如果你不知道数据库名咋办呢？ SHOW DATABASES就可以了。
选择数据库之后就可以对数据库里的表进行操作了，使用SHOW TABLES就可以知道有哪些表了。
如果要新建一个数据库，使用CREATE DATABASE db_name就可以创建一个名为db_name的数据库了。
